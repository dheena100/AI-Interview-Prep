# AI Interview Preparation Platform 🚀

A Flask-based AI-powered interview preparation system that:
- Generates interview questions
- Evaluates answers using AI
- Extracts resume skills
- Gives performance score

## Tech Stack
- Python Flask
- MySQL
- HTML, CSS, Bootstrap
- AI (OpenAI / Gemini)

## Features
- Resume upload + skill extraction
- AI mock interview
- Score evaluation dashboard
- Feedback system
